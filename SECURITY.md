# Security Policy

## Supported Versions

Security updates are provided for the latest released version of Brew-TUI.

## Reporting a Vulnerability

Please use GitHub private vulnerability reporting for this repository.

Do not open a public issue for suspected vulnerabilities, license bypasses, credential exposure, or supply-chain concerns. Include:

- A concise description of the issue
- Reproduction steps or a proof of concept
- Impact and affected version, if known
- Any relevant logs with secrets removed

I will review valid reports as soon as practical and coordinate a fix before public disclosure.

## Security Scope

In scope:

- Brew-TUI CLI behavior
- BrewBar installer and update flow
- License storage and activation flow
- Dependency and release artifact integrity

Out of scope:

- Vulnerabilities in Homebrew itself
- Issues requiring physical access to a user's machine
- Social engineering
- Denial-of-service against public third-party services
